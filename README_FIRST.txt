HB STORE LEDGER PRO - Build Notes

If Visual Studio shows CS2001 (source file could not be found):

1. Close Visual Studio.
2. Delete the entire extracted project folder.
3. Extract the ZIP again to a local path NOT in OneDrive/Dropbox.
   Example: C:\HB\HB_STORE_LEDGER_PRO\
4. Open ManagerPaperworkSystem.sln from that folder.

This is an extraction/sync issue, not a code issue.
